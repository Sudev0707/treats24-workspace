Treats24 Workspace - Internal Project Management Application

Purpose

Treats24 Workspace is a private, internal project management and issue
tracking application for the Treats24 team. The application is not
public and is intended to manage software development, tasks, bugs,
releases, documentation, and team collaboration.

Goals

-   Track progress of all Treats24 applications.
-   Manage projects, tasks, and issues similar to Jira.
-   Maintain documentation similar to Notion.
-   Monitor team productivity and releases.
-   Keep costs at ₹0 initially.

Applications Managed

1.  User App
2.  Restaurant Partner App
3.  Admin Dashboard
4.  Backend APIs
5.  Infrastructure

Technology Stack

Frontend: - React - TypeScript - Tailwind CSS - React Query - Zustand
